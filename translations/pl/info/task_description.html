Zaczynamy serię związaną z hasłami. Każda następna misja oparta jest na poprzedniej i jest od niej trochę bardziej skomplikowana.

Ta misja polega na skonstruowaniu funkcji weryfikującej hasło.

Warunek poprawności hasła:

  • długość hasła powinna być większa niż 6.

Wejście: String.

Wyjście: Bool.

Przykład:

{% if interpreter.slug == "js-node" %} 
isAcceptablePassword('short') == false
isAcceptablePassword('muchlonger') == true
isAcceptablePassword('ashort') == false
{% else %} 
is_acceptable_password('short') == False
is_acceptable_password('muchlonger') == True
{% endif %}

Do czego jest używane: W formularzach do weryfikacji hasła.
