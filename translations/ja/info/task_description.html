あなたはパスワードシリーズの始まりにいます。すべてのミッションは前のミッションに基づいており、進めるごとに少しずつ複雑なものになっていきます。

このミッションでは、パスワードの検証機能を作成する必要があります。

以下が検証条件です:

  • 長さは6文字よりも大きくなければならない。

入力: 文字列

出力: 真偽値

例:

{% if interpreter.slug == "js-node" %} 
isAcceptablePassword('short') == false
isAcceptablePassword('muchlonger') == true
isAcceptablePassword('ashort') == false
{% else %} 
is_acceptable_password('short') == False
is_acceptable_password('muchlonger') == True
{% endif %}

どのように使用されるか: パスワードの確認フォームのため。また、これはタスクがどのように評価されるのかを知るためにも良いでしょう。
